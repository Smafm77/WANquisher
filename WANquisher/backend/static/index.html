Help / Info

This WAN Emulator spins up one server/router with multiple ports and matching clients. Use presets and per-field inputs to apply delay, jitter, loss, BER, rate limiting, and queue limits to any card.

  • Setup: choose number of ports and create the lab. Destroy removes all lab containers & networks.
  • Port overview: cards are split into Clients (top) and Ports (bottom). Change values and click “Apply” per card or “Apply changes” to batch.
  • Presets: pick a preset and use “Apply to all”, “Apply to all clients”, or “Apply to all ports”.
  • Pings: clients have “Ping server”; ports have “Ping client”. Use ∞ for continuous and Stop to end. RTT is shown per line.
  • Save/Load: on Setup, save current card settings to a JSON file or load a JSON to restore values onto matching cards.
  • Lab status: shows containers, interfaces, and networks in the current lab.

Tip: After loading a JSON, review the values then click “Apply changes” to enforce them in the lab.
